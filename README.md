# Healthcare Patient Analytics Dashboard

## Overview
This project analyzes 55,500 healthcare records using Microsoft Excel to identify patterns in patient demographics, medical conditions, treatment costs, healthcare utilization, and clinical outcomes.

## Objectives
* Analyze patient demographics
* Explore disease distribution patterns
* Compare treatment costs across conditions
* Evaluate admission patterns
* Build an interactive healthcare dashboard

## Tools Used
* Microsoft Excel
* Pivot Tables
* Pivot Charts
* VLOOKUP
* Conditional Formatting
* Dashboard Development

## Dataset
Source:
Healthcare Dataset by Prasad22 (Kaggle)

Original Dataset:
https://www.kaggle.com/datasets/prasad22/healthcare-dataset

## Key Metrics
* Total Patients: 55,500
* Average Patient Age: 51.54 Years
* Average Length of Stay: 15.51 Days
* Average Billing Amount: ₹25,539.32
